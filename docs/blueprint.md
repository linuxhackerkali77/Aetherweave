# **App Name**: AetherDash

## Core Features:

- Real-time Chat: Direct messaging and group chats with real-time updates using WebSockets.
- Notes Manager: Rich text editor for creating and managing notes.
- To-Do Taskboard: Kanban-style task management board for organizing tasks.
- File Storage: Upload, preview, and delete files.
- AI Assistant: Context-aware AI assistant powered by LLM with memory tool for improved support.
- Theme Customization: User controls for customizing the dashboard theme, including a neon intensity slider.
- Activity Feed: Aggregated activity feed for staying up-to-date on all app modules.

## Style Guidelines:

- Primary color: Neon Cyan (#00FFFF) for a vibrant, futuristic look.
- Background color: Dark charcoal gray (#1A1A1A) to provide contrast and a cyberpunk aesthetic (desaturated neon cyan).
- Accent color: Electric Purple (#BF00FF) for highlights and interactive elements, creating contrast.
- Body and headline font: 'Space Grotesk', a sans-serif with a computerized, techy feel. Use 'Inter' for longer text passages.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use neon-style icons to match the cyberpunk aesthetic.
- Glassmorphism effects with frosted glass backgrounds for panels.
- Subtle animations such as neon glows, hover pulse effects, and animated cyber lines.
